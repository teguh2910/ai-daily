Claude Code Artifacts Ubah Sesi Coding Jadi Dasbor Langsung

Anthropic merilis fitur Artifacts di Claude Code untuk pelanggan Team dan Enterprise, memungkinkan dasbor interaktif yang diperbarui otomatis saat sesi coding berlangsung.

Anthropic Claude Code Artifacts Kecerdasan Buatan Perangkat Lunak

Anthropic resmi memperkenalkan Artifacts di Claude Code pada 18 Juni 2026. Fitur beta ini tersedia untuk pelanggan paket Team dan Enterprise, dan mengubah sesi coding menjadi dasbor bersama yang diperbarui secara langsung.

Berbeda dengan sesi terminal biasa, Artifacts memungkinkan Claude Code menghasilkan halaman web interaktif dari hasil kerjanya. Tim engineering dapat melihat visualisasi data, log proses, hingga walkthrough pull request dalam satu tampilan yang otomatis mengikuti progres agen. Saat Claude Code menulis kode baru, dasbor ikut berubah tanpa harus menyalin output ke tempat lain. Setiap anggota tim yang membuka halaman Artifacts akan melihat versi yang sama, sehingga diskusi soal insiden tidak lagi tercecer di channel chat yang berbeda.

Pembaruan ini menjawab salah satu keluhan terbesar pengguna AI coding agent: sulitnya membagikan konteks kerja antar anggota tim. Manajer bisa memantau insiden secara langsung, sementara rekan satu tim memberi masukan lewat komentar pada dasbor yang sama. Anthropic juga menambahkan dukungan untuk menghubungkan sumber data eksternal, sehingga dasbor dapat menampilkan metrik produksi, hasil uji otomatis, dan status deployment secara real time.

Dampaknya terasa di tiga area. Pertama, proses code review menjadi lebih cepat karena reviewer tidak perlu meniru lingkungan kerja secara manual. Kedua, onboarding anggota tim baru terasa ringan karena dasbor Artifacts bisa menjadi dokumentasi hidup yang selalu mengikuti perubahan kode. Ketiga, respons terhadap insiden produksi meningkat karena setiap langkah mitigasi yang diambil Claude Code langsung tampil di dasbor yang bisa diawasi bersama.

Untuk saat ini Artifacts masih dalam tahap beta dan hanya dapat diakses oleh akun Team serta Enterprise. Langkah ini mempertegas posisi Anthropic di pasar agen pemrograman yang semakin ramai, bersaing dengan Cursor dan GitHub Copilot. Detail lengkap pengumuman tercantum dalam laporan VentureBeat dan catatan rilis Releasebot untuk Claude.
